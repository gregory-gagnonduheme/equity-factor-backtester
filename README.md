# Equity Factor Backtester

A factor-based equity screener and backtesting engine for U.S. large-cap 
stocks, built in Python. Constructs momentum, value, and volatility signals 
on the S&P 500 universe and evaluates long-only and long/short strategies 
against a benchmark.

## Motivation

Built as a hands-on project to deepen Python skills in a finance context, 
bridging quantitative research experience (see supervised project on 
nonlinear factor dynamics in real assets) with systematic equity strategy 
implementation.

## Features

- [ ] S&P 500 data pipeline via yfinance
- [ ] Factor construction: momentum, value, volatility
- [ ] Monthly rebalancing backtest engine
- [ ] Risk metrics: Sharpe ratio, max drawdown, VaR
- [ ] Performance tearsheet vs S&P 500 benchmark
